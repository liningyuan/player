# player
基于Web Audio API的文本格式的播放器，用于播放音乐与数学课的程序输出。
以空格、换行分隔音符。正整数表示以半音为单位的音高，60为C4。0表示休止符，-1表示延续前一个音。默认120bpm,4/4拍，默认音符长度1个8分音符，可以逗号连接八分音符为单位的长度或用一串等号表示长度。
示例：蜗牛与黄鹂鸟
